Most UI debates
are not really
UI debates.

One person says "popup."
Another says "modal."
Someone else means "drawer."

Everyone thinks they are aligned, but they are discussing different things.

Popup
Small, lightweight,
usually non-blocking.
Quick message.
Doesn't interrupt.
Modal
Displays content in
a layer above.
Requires focus.
Blocks background.
Drawer
Slides in from the edge,
keeps context visible.
Supports exploration.
Keeps user in flow.

Different words. Different mental models.
That's why clarity is important.

Terminology
matters.

A component is not just
a shape on the screen.

It carries a decision.

It interrupts (or doesn't)
Some components demand
focus. Others don't.
It preserves (or changes) context
Context shapes how easily users
can return to where they were.
It guides attention
Different components pull focus
in different ways.
It compares information
Some components are for quick
scans, others for deep comparison.
It creates a next step
Every component influences
what the user does next.

Choosing the component
means choosing the behavior.
