# CC Carousel

A LinkedIn carousel for **Component Compass** — a UI decision guide for product builders.

The carousel introduces the project with the theme *"Good UI decisions start with clear language"* and walks through the component reference for PMs, developers, and designers.

## Contents

| File | What it is |
|------|------------|
| `CC LinkedIn Carousel v2.html` | The carousel itself — open in a browser to view all slides |
| `component-compass-logo-lockup.png` | Component Compass logo lockup |
| `uploads/` | Images used in the carousel slides |

## Viewing it

Open `CC LinkedIn Carousel v2.html` directly in a browser, or serve the folder locally:

```bash
npx serve .
```

Then visit the printed URL and open the carousel HTML.

## Exporting for LinkedIn

The carousel is designed to be exported as a PDF (one slide per page) for upload to LinkedIn. Render the HTML to PDF, then post the PDF as a document on LinkedIn to get the swipeable carousel.
